# Jeong 앱 데이터 출처 및 리뷰 구분 정책

> 장소 데이터와 리뷰의 출처를 명확히 구분하여 신뢰도 높은 정보 제공

---

## 1. 장소 데이터 출처 구분

### 두 가지 데이터 소스

| 구분 | 소스 | 아이콘/뱃지 | 설명 |
|------|------|:---:|------|
| **공식 데이터** | 네이버 플레이스 API 등 | 🏢 `Official` | 외부 플랫폼에서 수집한 검증된 데이터 |
| **사용자 등록** | Jeong 앱 사용자 | 👤 `Community` | 사용자가 직접 등록한 장소 |

### 장소 카드 UI 표시 방식

```
┌─────────────────────────────────────────┐
│  📍 을지로 노가리 골목                     │
│  ⭐ 4.3  ·  🏢 Official                  │  ← 출처 뱃지
│  한식 · 종로구 · 도보 5분                   │
│                                         │
│  "현지인들이 퇴근 후 즐겨 찾는 골목"         │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│  📍 숨은 떡볶이집                          │
│  ⭐ 4.8  ·  👤 Community                 │  ← 사용자 등록 뱃지
│  분식 · 마포구 · 도보 12분                  │
│                                         │
│  "한국 친구가 데려간 곳, 관광객 0%"          │
│  등록자: @traveler_mike                   │  ← 등록자 표시
└─────────────────────────────────────────┘
```

### 데이터 소스별 필드 매핑

| 필드 | 공식 데이터 (네이버 등) | 사용자 등록 |
|------|:---:|:---:|
| 장소명 | ✅ 자동 | ✅ 직접 입력 |
| 주소/위치 | ✅ 자동 | ✅ 지도 핀 또는 주소 검색 |
| 카테고리 | ✅ 자동 | ✅ 선택 |
| 영업시간 | ✅ 자동 | ⬜ 선택 입력 |
| 전화번호 | ✅ 자동 | ⬜ 선택 입력 |
| 메뉴/가격 | ✅ 자동 (있는 경우) | ⬜ 선택 입력 |
| 대표 사진 | ✅ 자동 | ✅ 필수 1장 이상 |
| 한줄 소개 | ❌ | ✅ 필수 (왜 추천하는지) |
| 외국인 팁 | ❌ | ⬜ 선택 (영어 메뉴 유무, 카드 결제 등) |

### 사용자 등록 장소 품질 관리

| 단계 | 내용 |
|------|------|
| **등록 조건** | 로그인 필수 (Hard Wall) |
| **필수 입력** | 장소명, 위치, 카테고리, 사진 1장, 한줄 소개 |
| **검증 프로세스** | 등록 즉시 공개, 신고 누적 시 비공개 전환 |
| **신뢰도 시스템** | 등록자의 기여 횟수·평판에 따라 신뢰도 뱃지 부여 |
| **중복 방지** | 위치 반경 50m 내 동일 카테고리 장소 중복 경고 |

## 2. 리뷰 출처 구분

### 세 가지 리뷰 탭 구조

```
┌─────────────────────────────────────────┐
│  을지로 노가리 골목                        │
│  ⭐ 4.3 (리뷰 127개)                     │
│                                         │
│  ┌──────┬──────────┬──────────┐         │
│  │ 전체  │ Jeong 리뷰 │ 외부 리뷰  │         │
│  │ 127  │    23     │   104    │         │
│  └──────┴──────────┴──────────┘         │
│                                         │
│  [필터: 최신순 ▼] [언어: 전체 ▼]           │
└─────────────────────────────────────────┘
```

| 탭 | 내용 | 뱃지 | 설명 |
|---|---|:---:|---|
| **전체** | 모든 리뷰 통합 표시 | - | 기본 뷰, 출처 뱃지로 구분 |
| **Jeong 리뷰** | 앱 사용자 직접 작성 | 💬 `Jeong` | 실제 방문 기반 리뷰 |
| **외부 리뷰** | 네이버 등 외부 플랫폼 | 🔗 `Naver` | 크롤링/API 수집 리뷰 |

### 리뷰 카드 UI

```
┌─────────────────────────────────────────┐
│  💬 Jeong Review                        │  ← 출처 뱃지

│  👍 12  💬 3                             │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│  🔗 Naver Review                        │  ← 외부 출처 명시
│  ⭐⭐⭐⭐  ·  네이버 사용자                 │
│  2026.02.15                             │
│                                         │
│  "직장인 회식 장소로 자주 갑니다.            │
```

### 리뷰 필터 옵션

| 필터 | 옵션 |
|------|------|
| **출처** | 전체 / Jeong 리뷰만 / 외부 리뷰만 |
| **정렬** | 최신순 / 평점 높은순 / 도움순 |
| **언어** | 전체 / English / 한국어 / 日本語 / 中文 |
| **평점** | 전체 / ⭐4+ / ⭐3+ |

## 3. 평점 체계

### 이중 평점 표시

```
┌─────────────────────────────────────────┐
│  을지로 노가리 골목                        │
│                                         │
│  종합 평점  ⭐ 4.3  (127개)               │
│  ├─ 💬 Jeong  ⭐ 4.6  (23개)             │  ← 앱 사용자 평점
│  └─ 🔗 외부    ⭐ 4.2  (104개)            │  ← 외부 플랫폼 평점
│                                         │
└─────────────────────────────────────────┘
```

| 항목 | 정책 |
|------|------|
| **종합 평점** | 전체 리뷰 가중 평균 (Jeong 리뷰에 약간 높은 가중치) |
| **Jeong 평점** | 앱 사용자 리뷰만으로 계산 |
| **외부 평점** | 외부 플랫폼 리뷰만으로 계산 |
| **최소 표시 기준** | 리뷰 3개 이상일 때 평점 표시, 미만이면 "리뷰 부족" |
| **가중치** | Jeong 리뷰 1.2배 (자체 생태계 활성화 유도) |

## 4. 외부 데이터 연동 정책

### 네이버 플레이스 데이터

| 항목 | 정책 |
|------|------|
| **수집 방식** | 네이버 Search API / Place API |
| **갱신 주기** | 주 1회 배치 업데이트 |
| **표시 의무** | 출처 "Naver" 뱃지 필수 표시 |
| **리뷰 번역** | 한국어 리뷰 → 영어/일어/중어 자동 번역 제공 |
| **저작권** | 네이버 API 이용약관 준수, 원문 링크 제공 |

| 소스 | 우선순위 | 활용 |

## 5. 데이터 모델 설계 (요약)

```dart
// 장소 데이터 출처
enum PlaceSource {
  official,   // 네이버 등 외부 API
  community,  // 사용자 등록
}

// 리뷰 데이터 출처
enum ReviewSource {
  jeong,      // 앱 내 사용자 리뷰
  naver,      // 네이버 리뷰
  google,     // Google 리뷰 (향후)
}

// 장소 엔티티
class Place {
  final String id;
  final String name;
  final PlaceSource source;       // 출처 구분
  final String? registeredBy;     // community일 때 등록자 ID
  final double? jeongRating;      // Jeong 리뷰 평점
  final double? externalRating;   // 외부 리뷰 평점
  final double? combinedRating;   // 종합 평점
  // ...
}

// 리뷰 엔티티
class Review {
  final String id;
  final ReviewSource source;      // 출처 구분
  final String? authorId;         // jeong일 때 사용자 ID
  final String content;
  final double rating;
  final String? originalLanguage; // 원본 언어
  final String? translatedContent; // 번역 내용
  final List<String> photoUrls;
  // ...
}

// 리뷰 필터
class ReviewFilter {
  final ReviewSource? source;     // null = 전체
  final String? language;         // null = 전체
  final double? minRating;
  final ReviewSortType sortBy;
}
```
