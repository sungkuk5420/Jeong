# Jeong (정) - 종합 제품 기획서

> 외국인을 위한 한국 로컬 정보 교환 플랫폼

---

## 1. 앱 비전 & 미션

### 비전
> **"Every foreigner in Korea finds their place through Jeong"**
> 한국의 모든 외국인이 정(情)을 통해 자신만의 장소를 발견한다

### 미션
외국인이 한국에서 현지인 수준의 정보에 접근하고, 서로의 경험을 나누어 더 풍요로운 한국 생활을 할 수 있도록 돕는다.

### 핵심 가치

| 가치 | 설명 |
|------|------|
| **Trust (신뢰)** | 검증된 데이터 + 실제 방문 리뷰로 신뢰할 수 있는 정보 |
| **Locality (로컬리티)** | 관광객용이 아닌, 현지인이 진짜 가는 곳 |
| **Accessibility (접근성)** | 언어 장벽 없이 누구나 쉽게 사용 |
| **Connection (연결)** | 외국인과 한국, 외국인과 외국인을 연결 |

```
┌──────────────────────────────────────────────────┐

│     REST API · WebSocket · JWT Auth               │

│                   Database                        │
│  PostgreSQL (메인) · Redis (캐시) · Elasticsearch  │
├──────────────────────────────────────────────────┤
│                   Storage                         │
│              AWS S3 / Cloudflare R2               │
├──────────────────────────────────────────────────┤
│                 External APIs                     │
│  Naver Search · Google Maps · Firebase Auth       │
│  DeepL/Google Translate · FCM                     │
├──────────────────────────────────────────────────┤
│               Infrastructure                     │
│  AWS (ECS/RDS/ElastiCache) 또는 Supabase          │
│  GitHub Actions CI/CD · Codemagic (iOS)           │
└──────────────────────────────────────────────────┘
```

### 핵심 DB 스키마

```sql
-- 사용자
users (id, email, nickname, avatar_url, nationality, locale, auth_provider)

-- 장소
places (id, name, address, lat, lng, category, source, registered_by,
        jeong_rating, external_rating, combined_rating, foreigner_tips)

-- 리뷰
reviews (id, place_id, user_id, source, rating, content,
         original_language, translated_content, photos[], likes_count)

-- 북마크
bookmarks (id, user_id, place_id, collection_id)

-- 컬렉션
collections (id, user_id, name, is_public)

-- 커뮤니티
posts (id, user_id, board_type, title, content, photos[],
       likes_count, comments_count)
```
